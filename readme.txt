программа определяет местоположение с сервера https://get.geojs.io/v1/ip/geo.json
и выдает текущую погоду  с сервера https://api.open-meteo.com/v1/forecast?latitude=52.52&longitude=13.41&current_weather=true


второй вариант определяет местоположение по браузеру фуницей navigator.geolocation.getCurrentPosition ,
результат погоды при этом отличается. 

есть онлайн страница: https://konstantinglazunov.github.io/Weather/
